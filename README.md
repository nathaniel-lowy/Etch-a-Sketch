# Etch-A-Sketch
To interact with this project, [check out its GitHub Pages deployment.](https://nathaniel-lowy.github.io/Etch-a-Sketch/)

A simple drawing tool, [based on the Odin Project's Etch-a-Sketch project.](https://www.theodinproject.com/lessons/foundations-etch-a-sketch)

This project includes a few different features, including black/white vs color toggling, adjustable tile counts, and darkening upon repeat hovering.
